This directory contains image resources for the Ticket Booking System application.

The following image files should be placed in this directory:
- ticket_icon.png - Main application icon
- ticket_logo.png - Logo for the login screen
- home_icon.png - Icon for the home menu item
- admin_icon.png - Icon for the admin menu item
- login_icon.png - Icon for the login menu item
- logout_icon.png - Icon for the logout menu item
- exit_icon.png - Icon for the exit menu item

You can use any suitable icons with these filenames, or create your own.
The application will still work without these images, but will look more professional with them.
